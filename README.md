# WeatherApp

Ứng dụng xem thông tin thời tiết: WeatherApp

Dữ liệu thời tiết được lấy tại: [weatherapi.com](https://weatherapi.com)
Ngôn ngữ: Kotlin
Kiến thức sử dụng:
- Room Database
- Retrofit2
- Livedata
- Recyclerview
- Viewpager2
- Navigation Drawer
- Glide
- Lottie animation files
